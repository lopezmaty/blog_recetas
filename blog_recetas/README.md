# Blog de Recetas

Es un blog de recetas donde una comunidad de personas puede registrarse, agregar recetas, editarlas, eliminarlas y ver la de otros usuarios para aprender de cocina

Pasos para instalacion:

1- python -m venv env
2- Activar entorno: env\Scripts\activate.bat (Windows)
3- pip install -r requirements.txt
4- python manage.py runserver
